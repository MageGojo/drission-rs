# 标配补齐 · 对标 Playwright / Puppeteer / DrissionPage 的通用浏览器能力

> 定位:本库护城河(验证码 OCR / 滑块 / 点选 / 反检测 / 补环境 / TLS 指纹)已超市面;
> 但相比 Playwright(PW)/ Puppeteer / DrissionPage(DP)还缺一批**通用浏览器标配**。本文件
> 把这些缺口分两批补齐,并记录各能力的后端可行性、CDP 命令落点与里程碑进度。
>
> 恢复上下文顺序:`docs/需求.md` → `docs/设计.md` → `docs/进度.md` → `docs/第三梯队.md` → 本文件。

## 范围与后端策略

**默认后端 = Chromium / CDP**,且本批能力**绝大多数是 Chromium 专有**(`Page.printToPDF`、
`Page.captureSnapshot(mhtml)`、`Emulation.setEmulatedMedia`、`Network.emulateNetworkConditions`、
`Emulation.setCPUThrottlingRate`、`Browser.grantPermissions`、`Emulation.setTouchEmulationEnabled`、
`Runtime.addBinding` 等)。Firefox/Camoufox 走 Juggler 协议,**没有这些命令**。

故策略:**本批全部在 CDP 后端实现**(默认后端,ROI 最高、机制完备);Camoufox 端只把"纯通用 JS 即可
实现"的少数项(`set_content` / localStorage / `wait` 对齐)列为**后续对齐**,其余 CDP 专有项不在 Juggler 端重复造。

## API 设计(对齐既有句柄风格)

沿用 `tab.set()` / `tab.wait()` 句柄与顶层 `tab.*` 方法两种风格:

```rust
// —— 第一批 ——
tab.save_pdf("out.pdf", &PdfOptions::default()).await?;     // 或 tab.print_to_pdf(&opts) -> Vec<u8>
tab.set_content("<h1>hi</h1>").await?;                       // 直接灌 HTML
tab.save_mhtml("page.mhtml").await?;                         // 或 tab.mhtml() -> String

tab.set().emulate_dark(true).await?;                         // prefers-color-scheme: dark
tab.set().emulate_media(Some("print"), &[("prefers-reduced-motion","reduce")]).await?;
tab.set().offline(true).await?;                             // 离线
tab.set().network_conditions(&NetworkConditions::slow_3g()).await?;
tab.set().cpu_throttling(4.0).await?;                       // 4x 降速
tab.set().grant_permissions("https://example.com", &["geolocation","clipboard-read"]).await?;

tab.set().local_storage_set("k","v").await?;  tab.set().local_storage_get("k").await?;
tab.set().session_storage_set("k","v").await?;

let t2 = tab.wait().new_tab(Some(d)).await?;                // 等弹窗/新标签 → 新 Tab
tab.wait().download_begin(Some(d)).await?;                  // 等下载开始
tab.wait().network_idle(0.5, Some(d)).await?;              // 网络空闲 500ms
tab.wait().ele_loaded("#x", None).await?;                  // cdp 端补齐(camoufox 已有)

// —— 第二批 ——
tab.set().device(&Device::iphone_13()).await?;             // UA+视口+DPR+mobile+touch 一把梭
tab.set().touch(true).await?;
let rec = tab.har_record().await?;  /* ... */  let har = rec.stop().await?;  har.save("t.har")?;
let _g = tab.expose_function("addRust", |args| Ok(json!(args.len()))).await?;  // window.addRust()
```
