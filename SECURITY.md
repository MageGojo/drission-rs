# 安全策略 / Security Policy

## 支持的版本

本项目处于活跃开发期,安全修复仅针对**最新发布版本**。

| 版本 | 支持 |
|---|---|
| 0.4.x(最新) | ✅ |
| 更早 | ❌ |

## 报告漏洞

**请不要**通过公开 issue 报告安全问题。

请优先使用 GitHub 的私密通道:仓库 **Security → Report a vulnerability**
(<https://github.com/MageGojo/drission-rs/security/advisories/new>)。
也可经 [apizero.cn](https://apizero.cn) 的联系方式私下告知。

报告时请尽量附上:受影响版本、复现步骤或 PoC、影响评估、可能的修复建议。

我们会尽快确认收悉,并在评估后协调修复与披露时间线。

## 使用须知

本项目仅供学习与合法、非盈利用途。请遵守目标站点的 `robots` 协议与当地法律法规,
不得用于任何违法、攻击骚扰或采集受保护数据的行为。详见 [`LICENSE`](LICENSE) 与 README 免责声明。
