# CDP 后端「每浏览器不同指纹」

> 关联里程碑:64。代码 `src/cdp/fingerprint.rs`;示例 `examples/cdp_fingerprint.rs`。
> 与 camoufox 的 `FingerprintPool`(里程碑 30,`src/pool/fingerprint.rs`)同源思路,面向**默认 CDP 后端**。

## 解决什么问题
同一台机器 / 同一出口 IP 上开**多个浏览器各注册/登录不同账号**时,若指纹一致会被站点**关联**(判定"同一人多开")。
本能力给每个浏览器一份**连贯且互不相同**的指纹,降低关联风险。

## 两种模式(过盾安全度 vs 差异化强度)

| 模式 | 入口 | 伪装 | 适用 |
|---|---|---|---|
| **同 OS 变体**(推荐默认) | `CdpFingerprint::random()` / `CdpFingerprintPool::generate(n)` | **保真** UA/platform/WebGL;只变 屏幕/时区/语言/硬件并发/内存 + canvas·audio 噪声 | 同机多账号反关联;**对 Turnstile 最友好**(无自相矛盾指纹) |
| **跨 OS 画像**(进阶) | `CdpFingerprint::random_persona()` / `CdpFingerprintPool::personas(n)` | **完整伪装** UA + platform + WebGL vendor/renderer + 屏幕 | 让 N 个浏览器像 N 台不同机器;**建议配套对应地区代理**,否则 WebGL 像素与跨 OS UA 可能被强检测识破 |

> 取舍依据 `docs/CDP过盾.md` 的"最小伪造最准"教训:撒的谎越少、越自洽,越不容易被识破。同 OS 变体不动 UA/WebGL,最稳。

## 改了哪些指纹面

- **启动级**(浏览器级、覆盖所有帧含 Turnstile 跨域子帧,走 `ChromiumOptions`):
  - `user_agent` → `--user-agent`
  - `locale` → `--lang` + `LANGUAGE`
  - `timezone` → `TZ` 环境变量
  - `window_size` → `--window-size`
- **深指纹**(进程内、JS 注入,走 `ChromiumOptions::init_scripts` → `Page.addScriptToEvaluateOnNewDocument`,在内置反检测脚本**之后**):
  - `navigator.platform` / `hardwareConcurrency` / `deviceMemory` / `languages` + `language`
  - `screen.{width,height,availWidth,availHeight}` / `colorDepth` / `pixelDepth` / `window.devicePixelRatio`
  - WebGL `getParameter`(`UNMASKED_VENDOR_WEBGL=37445` / `UNMASKED_RENDERER_WEBGL=37446`)hook
  - **canvas / audio 噪声**:`getImageData`/`toDataURL` 每画像确定性 ±1 微噪 + `AnalyserNode.getFloatFrequencyData` 微噪。
    每次取数前把 PRNG 重置到画像种子 → **同画像跨调用稳定、跨画像各异**(这是同机多账号 canvas/audio 反关联的关键)。

## 用法

### 单浏览器
```rust
use drission::cdp::{ChromiumBrowser, ChromiumOptions, CdpFingerprint};

let fp = CdpFingerprint::random();                    // 同 OS 变体(Turnstile 友好)
let opts = fp.apply_to_options(ChromiumOptions::new().headless(true));
let browser = ChromiumBrowser::launch(opts).await?;
```

### 并发池(每 worker 一份指纹)
```rust
use drission::cdp::{ChromiumOptions, ChromiumPool, ChromiumPoolOptions, CdpFingerprintPool};

let pool_fp = CdpFingerprintPool::generate(5);
let worker_opts = pool_fp.worker_options(&ChromiumOptions::new().headless(true));
let browsers = ChromiumPool::launch(
    ChromiumPoolOptions::new().size(5).worker_options(worker_opts)
).await?;
```

### 自定义画像(builder)
```rust
let fp = CdpFingerprint::new()
    .user_agent("Mozilla/5.0 (Windows NT 10.0; Win64; x64) … Chrome/133.0.0.0 Safari/537.36")
    .platform("Win32")
    .locale("en-US").timezone("America/New_York")
    .window_size(1920, 1080).screen_size(1920, 1080)
    .hardware_concurrency(12).device_memory(16)
    .webgl("Google Inc. (NVIDIA)", "ANGLE (NVIDIA, NVIDIA GeForce RTX 3060 …, D3D11)")
    .canvas_noise(0xC0FFEE);
```

## 验证示例
```
cargo run --example cdp_fingerprint              # 默认无头、3 个、同 OS 变体
N=5 cargo run --example cdp_fingerprint          # 5 个
PERSONA=1 cargo run --example cdp_fingerprint    # 跨 OS 完整画像
HEADFUL=1 cargo run --example cdp_fingerprint    # 有头可视
```
实测(本机 mac,Apple M4):同 OS 模式各浏览器 canvas#/时区/屏幕/硬件各异、UA/WebGL 保真;
跨 OS 模式 3 个浏览器分别呈现 Windows + NVIDIA RTX3060 / Intel UHD630 / AMD RX580。

## 注意
- **过 Turnstile 仍建议有头**(无头难过,见 `docs/CDP过盾.md`)。指纹差异化与过盾是两件事:本能力解决"多账号反关联",不替代过盾。
- 跨 OS 画像若不配代理,出口 IP 与时区/语言/UA 的国别不一致会**降可信度**;同 OS 变体不改 IP 相关强信号,最稳。
- persona 表里的 Chrome 主版本会随时间变旧,跨 OS 模式需要时自行更新 `PERSONAS`。
